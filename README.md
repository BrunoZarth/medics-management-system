# Features

<b>Back end:</b>

* Java
* Medic Management CRUD 
* RESTful API
* Clean code principles
* Maven
* Spring Boot
* Spring Data JPA
* Spring Web
* Spring Devtools
* Spring Starter Validation
* MySQL Connector   
* SQL Scripts to populate DB
* H2 Database (for unity tests)
* Documentation via Swagger
* TDD (Test Driven Development)
* Unity tests (JUnit, Mockito)
* Integration tests (Spring Boot Test)

<b>Front end:</b>

* Angular/Typescript
* Bootstrap
* External API Connection (post office API to consult adress)
* Find, Insert, Upgrade, Delete

<b>DevOps:</b>

* MySQL database
* Docker / Docker Compose
* Backend/frontend/database auto generated containers 
* Run the app locally with only one simple command (docker-compose up)

<br>

# Usage

<b>Using this APP is very simple:</b>

1) Download the project
2) Run docker compose command inside "medics-management-system": 

        docker-compose up --build
    
    <i>it will run all tests; generate images of the frontend, database and backend app; and then, run the respective containers.</i>
    <br><br>

    
3) Access backend documentation via Swagger:

        http://localhost:8080/swagger-ui/index.html#

4) Access app frontend:

        http://localhost:4200/
