# medics-management-system

usage:

1) Download project

2) Run command inside "medics-management-system": 
    - docker-compose up --build
    
it will run all tests from spring boot back-end, then, if all passes, generate his image and execute it connecting to a generated and executed image of MySql database.    
    
3) access documentation via swagger:
    - http://localhost:8080/swagger-ui/index.html#
